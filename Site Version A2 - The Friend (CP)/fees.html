Wellness Packages

Pricing is discussed during the consultation based on which package you're interested in and the duration (4, 6, or 12 weeks).

Why Private Pay

I chose private pay because it lets me work this way. No insurance requirements. No session limits. No pressure to see high volume. It means every person I work with is there because they chose to be—not because their insurance happened to cover it.

It also means I can work with you flexibly. If you need extra support between sessions, I can offer that. If we need to adjust our schedule, we can. If you need longer sessions sometimes, we can talk about it.

This model isn't for everyone. But for people who are ready to invest in themselves and want to work with someone who can really show up, it works.

Questions About Payment

If cost is a barrier, we can talk about it. Some people do a mix of therapy and packages. Some people start with a package and move into ongoing work. The point is to figure out what's actually sustainable for you.

Get in touch if you'd like to discuss options.
